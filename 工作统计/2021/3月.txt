postcss-px2rem-exclude 增加大屏适配并调整标准版兼容样式
技术点：
```
// 1.大屏版样式需要适配
// postcss.config.js 代码中加入exclude过滤掉element-ui的样式
module.exports = {
  plugins: {
    autoprefixer: {},
    'postcss-px2rem-exclude': {
      remUnit: 32,
      exclude: /node_modules|style/i,
    },
  },
}
// 2.标准版和大屏版颜色需要动态切换
// 问题1.不能刷新页面进行动态改变canvas里面的变量
// 问题2.模版颜色分为2套。如何组件式配置,以后还需要考虑多套颜色模版复用
```
